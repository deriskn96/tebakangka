Cara Menggunakan di GitHub Pages:

  1. Ganti URL tujuan pada variabel targetURL dengan URL yang Anda inginkan
  2. Simpan kode berikut sebagai index.html di repository GitHub Pages Anda
  3. Commit dan push perubahan
  4. Halaman Anda akan otomatis redirect ke URL tujuan
<!DOCTYPE html> <html lang="id"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Redirect Instant</title> <script> // URL tujuan redirect - GANTI DENGAN URL ANDA const targetURL = "https://website-tujuan-anda.com"; // Redirect segera setelah halaman dimuat window.onload = function() { window.location.href = targetURL; }; // Redirect secepat mungkin (sebelum halaman selesai dimuat) window.location.replace(targetURL); </script> <style> body { font-family: Arial, sans-serif; text-align: center; padding: 50px; background: #f9f9f9; } .loader { width: 50px; height: 50px; margin: 20px auto; border: 5px solid #f3f3f3; border-top: 5px solid #3498db; border-radius: 50%; animation: spin 1s linear infinite; } @keyframes spin { 0% { transform: rotate(0deg); } 100% { transform: rotate(360deg); } } </style> </head> <body> <div class="loader"></div> <h1>Redirect Instant</h1> <p>Anda akan diarahkan ke website tujuan.</p> <p>Jika tidak bekerja, <a href="https://website-tujuan-anda.com">klik di sini</a>.</p> </body> </html>
